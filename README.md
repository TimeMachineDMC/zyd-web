
# Justitia | 债优盾：工业级法财穿透研判引擎

## 项目简介 | Project Introduction

⚖️ **Justitia (Justitia's Eye)** > **基于 DeepSeek-V3.2 推理链与 BGE-M3 RAG 的法财一体化穿透研判系统**
> *Industrial-grade Law-Finance Integrated Intelligence Engine based on DeepSeek-V3.2 & BGE-M3 RAG*

**Justitia** 是一款专为复杂法律风险与财务穿透设计的深度研判引擎。它突破了传统 AI 仅能处理文本的局限，通过**双核驱动技术**（SQLite 财务穿透 + Chroma 法律卷宗 RAG），实现了对企业经营异常与法律违规风险的深度耦合分析。系统具备 2026 年夏季的时间锚点认知，并内置了针对 2025-2026 年最新司法解释与判例的语义检索能力。

**Justitia** is an industrial-grade intelligence engine designed for complex legal risks and financial penetration. It integrates **Dual-Core Data Driving** (SQLite financial analysis + Chroma legal RAG) to achieve deep coupled analysis of corporate operational anomalies and legal compliance risks.

## 项目架构 | Architecture Highlights

* **推理大脑 (Reasoning Brain)**: 基于 `DeepSeek-Reasoner` 构建，具备完整的思维链（Thought Process）展示，确保每一项法律研判逻辑透明、可追溯。
* **语义检索 (Semantic Retrieval)**: 采用 `BGE-M3` 向量模型对 2024-2026 年法律知识库进行高维度嵌入，支持长文本语义对标。
* **财务穿透 (Financial Penetration)**: 自动化 SQL 生成与执行引擎，能够直接从关系型数据库中提取经营指标并转化为研判证据。
* **工业级 UI (Industrial UI)**: 适配深色模式的工业风格界面，支持流式思维链渲染与呼吸灯交互反馈。
